# pro100_gym


## Как работаем с репой

### 0. Получили задачу в YouGile и берем ее slug
<img width="561" height="328" alt="image" src="https://github.com/user-attachments/assets/30fd75d6-0c2a-4488-a8f8-dc389f6dee49" />

### 1. Прежде чем делать задачу, создаем git-ветку с названием в виде слага

```bash
git checkout -b PRO-8
```

### 2. Делаем задачу, коммитим в созданную ветку. Название коммитов начинаем со слага

```bash
git commit -m 'PRO-8: добавил фичу'
```

### 3. Закончили делать задачу и убедились, что все работает - пушим ветку

```bash
git push --set-upstream origin PRO-8
```

### 4. Делаем ПР из этой ветки в main. В YouGile переводим задачу в "Тестируются"
### 5. Просим коллег поревьюить и потестить выполненную задачу
